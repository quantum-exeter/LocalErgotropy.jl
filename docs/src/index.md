# LocalErgotropy

Documentation for [LocalErgotropy](https://github.com/quantum-exeter/LocalErgotropy.jl).

`LocalErgotropy` is a package to compute the local ergotropy of a quantum
system. The ergotropy is obtained via numerical optimisation (gradient descent
over the unitary manifold).
